# llm-quest

Companion hands-on notebooks for the **LLM Quest** theory curriculum on [learningmap.net](https://learningmap.net).

Goal: turn every lesson in `llm-quest-theory/` into an interactive Jupyter notebook — learners read the theory, then open the notebook to *run code, observe results, and do exercises hands-on*.

## Layout at a glance

```
llm-quest/
├── llm-quest-theory/      # Theory content (.mdx) — READ-ONLY, do not edit (see learningmap.net)
│   ├── level-0/ … level-9/
├── notebooks/             # Practice notebooks (generated by Agents)
│   ├── level-0/ … level-9/
├── docs/                  # Project docs (read before building) (private)
│   ├── ARCHITECTURE.md
│   ├── CURRICULUM.md
│   ├── NOTEBOOK_SPEC.md
│   ├── AGENT_WORKFLOW.md
│   └── REQUIREMENTS.md
├── requirements.txt       # Python dependencies
├── CLAUDE.md              # Entry point for AI Agents
└── README.md
```

## For learners
1. Set up Python 3.10–3.12
```bash
conda create -n llmquest python=3.10
```
2. Install dependencies: `pip install -r requirements.txt`
3. Open the notebook for the lesson you want, e.g. `notebooks/level-0/0-1-variables-functions.ipynb`
4. Read the theory on learningmap.net for the background, then run the notebook cell by cell.
