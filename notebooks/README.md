# notebooks/

This folder contains the hands-on notebooks, mapped 1-to-1 with the theory files in `llm-quest-theory/`.

See [../docs/CURRICULUM.md](../docs/CURRICULUM.md) for the full list.

## Status

Each `level-N/` subfolder has its own `README.md` listing which lessons already have a notebook. Agents update that file whenever they add or modify a notebook.

## Quick start

```bash
jupyter lab notebooks/level-0/0-1-variables-functions.ipynb
```

If the kernel is not set up yet, see [../docs/REQUIREMENTS.md](../docs/REQUIREMENTS.md) §3.

## Language policy

All notebook text (markdown, comments, string labels, docstrings) is written in **English only**. See [../docs/NOTEBOOK_SPEC.md](../docs/NOTEBOOK_SPEC.md) for the full spec.
